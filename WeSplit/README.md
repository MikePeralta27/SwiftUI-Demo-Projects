# WeSplit

A SwiftUI demo that splits a restaurant check: enter the bill amount, pick how many people are sharing, choose a tip percentage, and see the grand total plus **amount per person**. Uses `NavigationStack`, `Form`, currency formatting for the current locale, a segmented tip picker, and keyboard focus with a **Done** toolbar button to dismiss the decimal pad.

## Requirements

- Xcode 15+ (SwiftUI)
- iOS 17+ (or the deployment target set in the Xcode project)

## Open in Xcode

1. Open `WeSplit.xcodeproj` in this folder.
2. Select a simulator or device and run (**⌘R**).
