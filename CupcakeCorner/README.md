# CupcakeCorner

Starter **SwiftUI** project for the **CupcakeCorner** / networking chapter: **`AsyncImage`** loading a remote image, with commented scaffolding for **iTunes search `Codable`** models and **`URLSession`** `async` decoding (ready to uncomment as you follow the course).

## Requirements

- Xcode 15+ with Swift 5.9+ (SwiftUI)
- iOS version as set in the Xcode project

## Open in Xcode

1. Open `CupcakeCorner.xcodeproj` in this folder.
2. Select a simulator or device and run (**⌘R**).

`AsyncImage` needs network access on device/simulator for the sample URL.

## Learn more

Follows **100 Days of SwiftUI** patterns: `AsyncImage` phases, optional `List` + `.task` networking, and `Codable` for JSON APIs.
